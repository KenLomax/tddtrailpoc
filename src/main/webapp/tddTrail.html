    2. ./install.sh -r platform_only initialize (Takes ~ 1 minutes)
    3. ./install.sh -r platform_only start (Takes ~ 1 minute)
    1. install.bat -r platform_only setup .
    2. install.bat -r platform_only initialize .
    3. install.bat -r platform_only start
  3. Open the back office at http://localhost:9001
  4. Bla bla about what they can see and do.
  

cd trailhome/TddTrail; mvn -Dtest=TrailSetupTest#testSuiteIsOnline test|| { echo "Test testSuiteIsOnline Unsuccessful"; }
